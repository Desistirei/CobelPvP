- Craftbukkit 1.7 (Spigot)
- Engine (Server Core - Spigot)
- Fanciful (JSON Messages)
- HCFactions (HCF Core)
- cHub (Hub Core)
- Practice (Practice Core)
- Queue-Bukkit (Queue Core)
- Redstone (not sure, probably some server handler)
